#### ⛱ About
> 一个基于Nuxtjs的半静态博客  
> https://blog.halberd.cn

#### 🏷️ Feature
* `Nuxtjs` 静态化打包
* 部署在 `cloudflare` ,自动监听修改并编译
* 使用 `github api` 在线修改

#### 🚀 How to
1. `git clone`
2. `git remote set-url origin 'your repo'`
3. 创建一个`cloudflare page`
4. `git push`
